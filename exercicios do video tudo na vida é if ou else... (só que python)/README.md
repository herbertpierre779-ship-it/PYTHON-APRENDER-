# 🐍 Exercícios de Lógica com Python: Estruturas Condicionais

Este repositório contém uma série de scripts práticos desenvolvidos em **Python** para exercitar o uso de estruturas condicionais (`if`, `elif`, `else`) e operadores lógicos (`and`, `or`). Os exercícios foram baseados em desafios de lógica de programação para iniciantes.

## 🚀 Tecnologias Utilizadas
* **Python 3.10+**
* Ambiente de execução (Terminal/CMD)

---

## 📂 Visão Geral dos Scripts

### 1. 🌡️ Verificador de Temperatura (`para_ve_se_ta_frio.py`)
Classifica o clima com base na entrada do usuário em graus Celsius.
* **Lógica:** Abaixo de 20°C (Frio), entre 21°C e 30°C (Normal) e acima de 30°C (Quente).

### 2. 🏅 Inscrição de Atletas (`verificado_escriçao_atleta.py`)
Um sistema de validação para competições olímpicas que utiliza múltiplas condições em uma única estrutura.
* **Requisitos:** Idade entre 18 e 25 anos **E** (Bom condicionamento físico **OU** Permissão médica).

### 3. ⏰ Cumprimento por Horário (`bomdia_depende_da_hora.py`)
Exibe uma saudação personalizada dependendo da hora informada.
* **Destaque:** O script demonstra duas formas de escrita: usando operadores lógicos explícitos (`and`) e a forma simplificada (encadeamento de comparações) do Python.

### 4. 🥖 Simulação de Compra (`compra_pao.py`)
Um pequeno sistema de caixa que verifica se o saldo do usuário é suficiente para realizar uma compra.
* **Funcionalidade:** Se aprovado, o sistema subtrai o valor do produto e exibe o troco atualizado.
