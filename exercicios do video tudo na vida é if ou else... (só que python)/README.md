# 🐍 Exercícios de Lógica com Python: Estruturas Condicionais

Este repositório contém uma série de **scripts práticos em Python** para exercitar estruturas condicionais (`if`, `elif`, `else`) e operadores lógicos (`and`, `or`).  
Os exercícios foram baseados em desafios de lógica de programação para iniciantes.

---

## 🚀 Tecnologias Utilizadas
- **Python 3.10+**
- Ambiente de execução: Terminal / CMD

---

## 📂 Scripts Disponíveis

### 1. 🌡️ [Verificador de Temperatura](para_ve_se_ta_frio.py)  
Classifica o clima com base na entrada do usuário em graus Celsius.  
- **Lógica:** Abaixo de 20°C → Frio, entre 21°C e 30°C → Normal, acima de 30°C → Quente.

### 2. 🏅 [Inscrição de Atletas](verificado_escriçao_atleta.py)  
Sistema de validação para competições olímpicas com múltiplas condições.  
- **Requisitos:** Idade entre 18 e 25 anos **E** (Bom condicionamento físico **OU** Permissão médica).

### 3. ⏰ [Cumprimento por Horário](bomdia_depende_da_hora.py)  
Exibe uma saudação personalizada dependendo da hora informada.  
- **Destaque:** Demonstra operadores lógicos explícitos (`and`) e forma simplificada de comparações.

### 4. 🥖 [Simulação de Compra](compra_pao.py)  
Sistema de caixa que verifica se o saldo do usuário é suficiente para realizar uma compra.  
- **Funcionalidade:** Subtrai o valor do produto e exibe o troco atualizado.
